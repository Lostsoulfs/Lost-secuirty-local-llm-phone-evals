# Model Provenance

Record every model, app build, runtime tool, or benchmark package used for local
phone evals. Do not commit model files, APKs, private logs, or generated reports.

## Required fields

| Field | Meaning |
| --- | --- |
| Name | Model, app, runtime, or tool name. |
| Source | Official URL or trusted release page. |
| Version | Version, tag, build, or release date. |
| License | License or use restriction summary. |
| Checksum | SHA256 or provider checksum when available. |
| Local path pattern | Ignored local path such as `models/<name>/`. |
| Notes | Device, quantization, runtime, or setup notes. |
